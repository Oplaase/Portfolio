\# Projet



Portfolio personnel développé en HTML, CSS et JavaScript vanilla.



Le site doit être moderne, fluide, minimaliste et responsive.



\# Objectifs



\- Design propre et professionnel

\- Bonne lisibilité

\- Animations légères

\- Responsive mobile/tablette/desktop

\- Code simple et maintenable



\# Stack



\- HTML5

\- CSS3

\- JavaScript vanilla

\- Pas de framework

\- Pas de jQuery



\# Structure du projet



\- index.html -> page principale

\- css/ -> tous les fichiers CSS

\- js/ -> scripts JavaScript

\- assets/ -> images et icônes



\# Couleurs



Palette principale :

\- bleu nuit

\- cyan doux

\- gris foncé

\- texte clair



Style visuel :

\- design moderne et minimaliste

\- ambiance développeur premium

\- contraste propre et lisible

\- effets lumineux subtils

\- éviter les couleurs flashy ou saturées



Toujours privilégier :

\- animations discrètes

\- ombres légères

\- spacing propre

\- interface élégante et moderne



\# Animations



\- Transitions fluides

\- Animations discrètes

\- Priorité aux performances

\- Utiliser CSS animations quand possible



\# Responsive



Le site doit être responsive pour :

\- mobile

\- tablette

\- desktop



Toujours utiliser :

\- flexbox

\- grid

\- media queries propres



\# Conventions CSS



\- Utiliser des classes claires

\- Éviter les IDs pour le style

\- Organiser le CSS par sections

\- Éviter les styles inline



\# Conventions JavaScript



\- JavaScript simple et lisible

\- Fonctions courtes

\- Commentaires uniquement si nécessaire

\- Éviter le code dupliqué



\# Important



\- Ne jamais casser le responsive

\- Préserver l'accessibilité

\- Préférer du code simple plutôt que complexe

\- Toujours garder un design cohérent



\# Structure du portfolio



Le portfolio doit contenir les sections suivantes :



\- Accueil 

\- À propos

\- Bloc de compétences 1 à 5 (marqué tel quel, ne pas détailler 2,3,4)

\- Bloc de compétences 6

\- Tableau de synthèse des compétences

\- Poursuite d'études

\- Contact
